# Change Log

## 0.1.0

- Initial read-only VS Code notebook MCP server.
- Added local Streamable HTTP MCP endpoint at `127.0.0.1:<port>/mcp`.
- Added notebook context, focused cell, visible cells, last executed cell, recent execution history, and cell output tools.
- Added output truncation, binary output metadata-only handling, and optional output exposure controls.
- Added Japanese README, VSIX packaging scripts, and release automation.
